L-System Generator

Documentation

L-System Generator is a web application for the procedural generation and visualization of plant growth and fractals using L-systems (Lindenmayer systems).

It is possible to create complex figures starting from initial parameters and observe their evolution in real-time through interactive animations.

1. How to generate a drawing?

L-Systems are defined by 4 parameters:

  • Axiom: the initial string from which generation starts.
  • Rules: how to transform the variables at each iteration.
  • Depth (Recursion Level): number of rule iterations. You can obtain complex drawings by increasing this value.
  • Angle (θ): angle between two consecutive lines.
Example: Tree branch drawing
Axiom: F
Rule for F: F[+F]F[-F]F
Depth: 4
Angle: 25
Starting Rotation: -90

2. Rules and Commands

+
Turn left by θ degrees
-
Turn right by θ degrees
[
Save current position and angle
]
Restore saved position and angle

Customizable Variables

Any other alphanumeric character (e.g., F, X, A) is a customizable variable.

You can assign a specific behavior to each variable:

  • Draw Line: Draws a line forward.
  • Move To: Moves forward without drawing (creates empty spaces).
  • Draw Dot: Draws a dot at the current position.
  • Do Nothing: Performs no movement.

3. Graphic Settings

You can customize the drawing appearance:

  • Movement associated with each variable.
  • Stroke color and width.
  • Initial rotation (useful for orienting trees upwards).
  • Canvas background color.

4. Visualization Features

  • Auto-Centering: Automatic centering and zooming of the drawing.
  • Animation: Show/hide or pause/resume drawing.
  • Fullscreen: Press the f key to view the drawing in fullscreen.
